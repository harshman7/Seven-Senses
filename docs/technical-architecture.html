Technical architecture

Reference notes for a future Next.js implementation (this site is static HTML).

1. Core tech stack

  • Framework: Next.js 16 (App Router) — Server Components for imagery, Client Components for motion.
  • Styling: Tailwind CSS v4 — Linen & Sage design tokens.
  • Animations: Framer Motion — scroll reveals, subtle parallax.
  • Email: Resend — invitation and concierge notifications.

2. Implementation strategy

Colors: linen (#F5F0E8), sage (#8A9E8C), charcoal (#1C1A17). Typography: Cormorant Garamond and Jost. Layout: max-w-7xl containers, responsive padding.

Motion: whileInView transitions (~1.2s), custom easing; sticky nav with backdrop-blur-md.

Performance: next/image with priority on heroes; blur placeholders on journal grid.

Inquiry: Server Actions + Resend for confirmations.
